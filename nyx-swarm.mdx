Use this page to run the **Nyx Vortex Dashboard** simulation.

You can adjust **Max agents**, **Branch chance**, and **Spawn delay**.

You can start, pause, resume, and reset each run.

The right panel shows the thought process events and run outcomes.

## Syntax languages used

This demo and its docs use these syntax languages:

- `HTML` for page structure in `nyx-swarm.html`
- `CSS` for dashboard styling in `nyx-swarm.html`
- `JavaScript` for simulation logic and interactivity in `nyx-swarm.html`
- `MDX` for the documentation wrapper page in `nyx-swarm.mdx`
- `YAML` for MDX frontmatter metadata in `nyx-swarm.mdx`
- `JSON` for site navigation and configuration in `docs.json`
